# 🔍 Detective Mystery: Solve the Heist

A command-line detective mystery game where you must interview suspects, collect evidence, and solve a diamond heist crime! 

**Perfect for:** Practicing Python, learning game logic, interactive storytelling, and problem-solving!

---

## 🎮 Game Overview

A priceless diamond has been stolen from the mansion! As a detective, you must:
- 🗣️ **Interview 5 suspects** and listen to their alibis
- 🔍 **Investigate the crime scene** and collect clues
- 📊 **Analyze the evidence** you've gathered
- 🎯 **Make an accusation** and catch the criminal!

**Challenge:** You only have **15 moves** to solve the case before time runs out!
